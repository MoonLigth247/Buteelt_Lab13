2. Технологийн багцын харьцуулалт

    (Project Stacks)Бие даалтын А хэсгийн 1-р ажил бол 3+ stack-ийг харьцуулж нэгийг сонгох юм.

   Stack 1: MERN (MongoDB, Express, React, Node.js)Давуу тал: Бүх түвшинд JavaScript ашигладаг тул сурахад хурдан. AI-д зориулсан сангууд болон жишээ код маш их байдаг.  Сул тал: Жижиг Task Tracker-т арай нүсэрдэх талтай. Өгөгдлийн сангийн бүтэцгүй (NoSQL) байдал нь дараа нь логик алдаа гаргахад хүргэж магадгүй.

   Stack 2: Next.js + Supabase + Tailwind CSSДавуу тал: Одоогоор хамгийн тренд болж буй stack. Supabase нь Authentication болон Database-ийг шууд шийдэж өгдөг тул хөгжүүлэлтийн хугацааг маш их хэмнэнэ.  Сул тал: Supabase-ийн зарим функцууд шинэчлэгдэх бүрт AI хуучин хувилбарын кодоор төөрөгдөх (hallucination) эрсдэлтэй.

   Stack 3: Laravel + Blade + MySQLДавуу тал: CRUD үйлдэл хийхэд зориулагдсан маш бат бөх бүтэцтэй. PHP болон Blade ашиглан маш хурдан хугацаанд ажилладаг вэб бүтээх боломжтой.  Сул тал: JavaScript-ийн орчин үеийн React/Next.js-тэй харьцуулахад UI-ийн интерактив байдал (real-time updates) бага зэрэг удаан байж магадгүй.

Эдгээр 3+ Stack AI-уудаас Claude Code-ийг сонгож байна учир нь гэвэл ашиглахад хялбар. Мөн түүнлэн Terminal-тай харьцаж болон дээр нь ажиллаж байсан туршлагатай учир.
